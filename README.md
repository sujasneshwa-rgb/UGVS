🎓 University Grievance Voting System (UGVS)

## 📖 Overview
The University Grievance Voting System (UGVS) is a web-based application that allows students to submit grievances and vote on existing issues. The system helps prioritize problems based on the number of votes, making it easier for authorities to address important concerns.



🚀 Features
- Submit grievances
- Vote on grievances
- One user can vote only once
- Automatic vote counting
- Admin panel for managing grievances


 🛠️ Technologies Used
- Python (Django)
- HTML, CSS, JavaScript
- SQLite / MySQL


⚙️ How to Run
1. Clone the repository
2. Open project folder
3. Run:
   python manage.py runserver
4. Open browser and go to:
   http://127.0.0.1:8000/



🗂️ Modules
- User Module (Login/Register, Submit Grievance, Voting)
- Admin Module (Manage grievances, view votes)



🔐 Security
- User authentication
- One vote per user restriction
